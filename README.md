# 📖 Overview

UCAB is a **full-stack cab booking platform** developed using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. The application enables users to register, log in, book cabs, manage bookings, and allows administrators to monitor and manage booking requests through a dedicated dashboard.

The project focuses on providing a simple, responsive, and user-friendly transportation solution while demonstrating full-stack web development concepts such as REST APIs, authentication, database integration, and deployment.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/deepthigovvala/cab-booking-mern.git
```

## Install Frontend

```bash
cd client
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## Install Backend

```bash
cd server
npm install
npm start
```

Runs on:

```
http://localhost:5000
```
